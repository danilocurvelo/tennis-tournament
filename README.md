# Tennis Tournament App

This project was initialized manually with **Next.js 15**, **TypeScript**, **Tailwind CSS 4**, **Prisma**, **ESLint 9**, **Prettier**, and **Jest**.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Run production build
- `npm run lint` - Lint files using ESLint
- `npm run test` - Run unit tests with Jest
- `npm run seed` - Seed the database with 128 mock players and first-round matches

## Setup

Install dependencies:

```bash
npm install
```

Generate Prisma client:

```bash
npx prisma generate
```

Run development server:

```bash
npm run dev
```
